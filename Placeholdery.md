## Gracz
```yaml
%afteroweklany_player_points% - Punkty
%afteroweklany_player_kills% - Zabójstwa
%afteroweklany_player_deaths% - Śmierci
```

## Klan
```yaml
%afteroweklany_clan_tag% - Tag klanu do którego należy gracz
%afteroweklany_clan_name% - Nazwa klanu do którego należy gracz
%afteroweklany_clan_kills% - Zabójstwa wszystkich członków klanu
%afteroweklany_clan_deaths% - Śmierci wszystkich członków klanu
%afteroweklany_clan_points% - Punkty wszystkich członków klanu
%afteroweklany_clan_members% - Członkowie klanu
%afteroweklany_clan_online_members% - Członkowie klanu którzy są online
```

## Topki graczy
```yaml
%afteroweklany_top_player_kills_1% - Topka zabójstw graczy (np. 1 gracz w topce)
%afteroweklany_top_player_deaths_1% - Topka śmierci graczy (np. 1 gracz w topce)
%afteroweklany_top_player_points_1% - Topka punktów graczy (np. 1 gracz w topce)
```

## Topki klanów
```yaml
%afteroweklany_top_clan_kills_1% - Topka zabójstw klanów (np. 1 klan w topce)
%afteroweklany_top_clan_deaths_1% - Topka śmierci klanów (np. 1 klan w topce)
%afteroweklany_top_clan_points_1% - Topka punktów klanów (np. 1 klan w topce)
```
